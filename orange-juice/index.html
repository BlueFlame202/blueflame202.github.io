Hello! This is my "academic website". A more accurate description is that I tend to use this as a place to unclutter my mind.

Unlike my other blog, Lemonade, this website is very lightweight, only using HTML, CSS, and the bare minimum JS needed to run MathJax, along with a tiny amount of dynamic content.

Check out some of the links in my navigation!
