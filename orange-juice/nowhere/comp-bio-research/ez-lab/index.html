E.Z. Lab Thoughts

Thoughts on CryoBoltz

General Thoughts:
  1. The idea of using Cryo-EM density maps to guide diffusion is a very cool way to add more data to the diffusion model at evaluation time. As always, I love optimal transport, so the idea of getting the point clouds via $k$-means to use Sinkhorn divergences was very cool.
  2. The global and local guidance phases are interesting. I was especially interested in how the local only benchmarks perform far worse than the global only ones, and often worse than Boltz-1 (except for the CH67 antibody).
  3. AlphaFold3 doing worse on outward state vs. inward state was interesting as well, whereas the opposite seems to be true of Boltz-1.
  4. Figure 4 was particularly interesting to see CryoBoltz fitting the CDR H3 loop better.
