Work

Work is done by a force on an object. Generally, the force is exerted by another object. We can specify work by specifying the object doing the work and the object which the work is done on, and also by specifying whether we are reerring to one particular force or the net forces. Work can then be computed as

$$W = \int \mathbf{F} \cdot d\mathbf{x}$$
